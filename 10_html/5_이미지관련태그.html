src 속성에 경로를 설정하여 이미지 파일을 불러온다.

자주 사용하는 속성들

alt 속성

화면낭독기는 이미지를 읽을 수 없기 때문에 텍스트를 대신 읽어줌
혹은 사진의 경로가 잘못되거나 이미지를 제대로 표현할 수 없는 경우 대체 텍스트의 용도로도 사용 가능.


width와 height 속성

사진의 높이와 너비 지정 가능
고정된 단위의 크기로도 지정 가능
가변 크기 단위 크기로도 지정 가능

고정 크기 단위: 화면 사이즈가 달라져도 크기 변화 X

가변 크기 단위: 화면 사이즈가 달라지면 그 사이즈에 따라 크기 변경

이미지 구역을 2개로 나눠 각각의 링크 설정
